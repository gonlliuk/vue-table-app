# [vue-table-app](https://vue-table-app.herokuapp.com)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run already built app on production
```
npm start
```

### Lints and fixes files
```
npm run lint
```
